1. INTRODUCTION

This project focuses on developing a system that allows users to upload documents such as PDF and text files. The system processes the uploaded files and extracts useful information from them. The main goal is to simplify document handling and provide quick insights.

--------------------------------------------------

2. OBJECTIVES

- To create a user-friendly file upload system
- To support PDF and text file formats
- To extract and analyze content from documents
- To provide summarized output for users

--------------------------------------------------

3. TECHNOLOGIES USED

- Programming Language: Python
- Framework: FastAPI
- Libraries:
  - PyPDF2 (for PDF reading)
  - LangChain (for text processing)
  - FAISS (for vector storage)

--------------------------------------------------

4. METHODOLOGY

Step 1: User uploads a document through the web interface  
Step 2: The system reads the file content  
Step 3: Text is divided into smaller chunks  
Step 4: Embeddings are created for analysis  
Step 5: User queries are processed using the stored data  

--------------------------------------------------

5. FEATURES

- File upload (PDF and TXT)
- Text extraction
- Question-answering system
- Fast processing

--------------------------------------------------

6. RESULTS

The system successfully uploads and processes documents. It provides accurate answers based on the uploaded content. The response time is efficient and suitable for real-time usage.

--------------------------------------------------

7. CONCLUSION

The project meets its objectives by enabling document upload and intelligent analysis. It can be further improved by adding support for more file types and better UI design.

--------------------------------------------------
